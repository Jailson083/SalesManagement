# Sales Management API

## Descrição
Esta é uma API para gerenciar vendas, implementada como parte de um teste técnico. A API permite criar, listar, atualizar e cancelar vendas.

## Tecnologias Utilizadas
- .NET 8
- Entity Framework Core
- Swagger (para documentação)
- xUnit (para testes unitários)
